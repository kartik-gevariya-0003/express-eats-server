## Getting Started

### Prerequisites

- Node
    - Install node and npm using `brew install node`.

### Installing

- Clone [repository](https://github.com/kartik-gevariya-0003/express-eats-server)
- Change current working directory to `express-eats-server` repository root folder
- Execute `npm install` to install dependencies
- Execute `npm start` to start backend server
